# Endpoint Monitoring Tools

Lightweight Python scripts for collecting basic endpoint and system health information.

These tools are designed to complement logging and SIEM pipelines by providing local system visibility.

## Included scripts
- Disk usage and disk health
- Memory usage
- System load
- Service status monitoring

## Intended use
- Endpoint visibility experiments
- Supporting SIEM ingestion pipelines
- Local troubleshooting and monitoring

## Tech
- Python
- Standard system libraries

## Notes
The scripts are intentionally simple and can be extended, scheduled, or integrated into larger monitoring workflows.
